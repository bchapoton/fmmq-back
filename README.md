# FMMQ Server
**NodeJS** v12.16.1

**MongoDB** MongoDB 4.2.5 Community

## Environment variables

|Name                 |Mandatory|default value|
|---------------------|:-------:|------------:|
|**FMMQ_mongo_db_uri**|yes      |N/A          |
|**FMMQ_private_key** |yes      |N/A          |
|**FMMQ_node_port**   |no       |8080         |


## Installation

```bash
yarn install
```

## Run

```bash
node ./bin/www
```
